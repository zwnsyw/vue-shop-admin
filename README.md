### 技术:

#### 	前端：

   1. Vue.js 

   2. Element-ui

   3. axios： 

   4. echarts： 图标

   5. nprogress： 进度条

   6. vue-quill-editor:  富文本编辑

   7. vue-table-with-treegrid: 表格树

   8. vue-router: 前端路由

   9. lodash: 深拷贝

   10. core: 跨域请求

#### 	后端：

    主要通过Node.js + MySQL提供API接口
    

### 功能：

1、用户登录验证

2、权限管理

3、商品管理

4、订单管理

5、数据统计
